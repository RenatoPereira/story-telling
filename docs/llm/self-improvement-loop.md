# Self Improvement Loop

## Required Loop For Every Task
1. Restate the task and define "done".
2. Read relevant context docs before coding.
3. Implement smallest viable change.
4. Run tests and lint.
5. Record lessons and follow-up actions.

## Definition Of Done
- Requirements implemented and validated.
- No new lint or type issues introduced.
- Tests added or updated where behavior changed.
- Decision log updated if architecture changed.
